   - 不要提交到公开Git

3. **不要使用大文件测试**
   - 测试视频控制在10MB以内
   - 避免超时错误

4. **不要慌张**
   - 遇到错误查日志
   - 文档里有解决方法

### 问题2：部署脚本报错
```
➡️ 检查：
   1. 是否安装了Azure CLI？
   2. 是否登录了Azure账号？
   3. 查看错误提示信息

```

### 问题3：Logic Apps不知道怎么创建
```
➡️ 打开：logic-apps-guide.md

   图形化操作说明
```

### 问题4：前端无法运行
```
➡️ 检查：

   2. 浏览器F12控制台有什么错误？
   3. Logic Apps是否运行成功？
```
