```html

- match-detail.js
- config.js
```

### styles.css
```css
功能：
- 所有页面的样式
- 响应式设计
- 动画效果
- 模态框样式
```

### script.js
```javascript
功能：
- 比赛列表加载
- 上传功能
- 文件处理
- 筛选功能

API调用：
- uploadMatch
- getMatches
```

### match-detail.js
```javascript
功能：
- 视频播放控制
- 战术标注（Canvas绘图）
- 评论管理
- 时间轴标记

API调用：
- getMatchById
- createAnnotation
- getAnnotations
- addComment
- getComments
```

### config.js
```javascript
配置内容：
- Azure Storage账户名

- 应用设置
```
